# Stitch 原型生成提示词（可直接粘贴）

请为我生成一个“中文Web端基金量化选基工具”高保真产品原型，面向理财初学者。目标市场是A股基金，覆盖场外基金和场内基金（ETF/LOF）。整体风格专业、简洁、可信，强调“可解释的推荐”。

【产品定位】
- 产品名：基金量化选基助手
- 用户：理财新手
- 核心价值：多因子评分 + 易懂解释 + 友好交互
- 重要声明：页面中显著展示“仅供参考，不构成投资建议”

【页面结构】
请生成以下页面并保持统一设计系统：
1) 首页/仪表盘
2) 选基页（核心）
3) 基金详情页
4) 基金对比页（2-5只）
5) 我的观察池页面

【首页/仪表盘需求】
- 顶部导航：首页、选基、对比、观察池、设置
- 概览卡片：今日推荐数量、高风险预警数、市场温度、最近更新日期
- “今日推荐Top10”表格：基金代码、名称、类型（场内/场外）、综合分、风险等级、操作按钮
- 风险提示模块：用醒目但不恐慌的方式展示
- 支持“风险偏好切换”：保守/均衡/进取（分段控件）

【选基页需求（核心）】
- 左侧为筛选器，右侧为结果列表
- 筛选项包括：
  - 场内/场外
  - 基金类别（宽基/行业/债券/混合）
  - 成立年限
  - 规模区间
  - 费率上限
  - 风险等级
  - 关键词搜索（代码/名称）
- 结果表支持排序（综合分、近1年收益、最大回撤、费率）
- 每行展示：代码、名称、类型、综合分、收益、回撤、费率、流动性标签、收藏按钮、查看详情按钮
- 顶部提供“因子权重模板选择”：保守/均衡/进取
- 提供“重置筛选”和“导出结果”按钮

【基金详情页需求】
- 头部：基金名、代码、类型、综合分、风险等级、收藏按钮
- Tab区：
  1. 表现：收益曲线 + 回撤曲线
  2. 因子分析：雷达图 + 分项评分条形图
  3. 成本与交易：费率、场内流动性、跟踪误差（若适用）
  4. 解释：推荐理由（加分项/扣分项）+ 风险提示
- 展示“综合评分公式说明”（可折叠）
- 显示“数据更新时间”

【对比页需求】
- 支持输入或从观察池选择2-5只基金
- 并排比较模块：
  - 综合评分对比
  - 收益/回撤曲线同图对比
  - 因子雷达图对比
  - 成本与流动性表格对比
- 提供“结论摘要卡片”：哪只更稳健、哪只弹性更高、适合何种风险偏好

【观察池页面需求】
- 卡片/表格混合展示收藏基金
- 显示评分趋势（小型sparkline）
- 显示预警状态（如回撤超阈值、评分下滑）
- 支持移除收藏、跳转详情、加入对比

【设计与交互要求】
- 语言：简体中文
- 风格：金融专业感 + 新手友好，避免花哨
- 颜色：主色建议蓝/青，风险用黄/橙/红分级
- 组件：使用现代企业级后台风格（卡片、数据表、分段控件、标签、抽屉）
- 图表：折线图、面积图、雷达图、条形图（保证可读性）
- 响应式：桌面优先，同时适配平板
- 可访问性：颜色对比合格、按钮与文字可读性高

【数据与状态】
- 先使用模拟数据（mock data）填充所有页面
- 包含空状态、加载状态、无结果状态、错误状态
- 每个关键操作给出反馈（toast/提示条）

【输出要求】
- 输出完整页面原型
- 组件命名清晰，便于前端开发映射
- 保持跨页面一致的设计系统（颜色、间距、字体、按钮样式）
