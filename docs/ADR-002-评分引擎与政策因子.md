# ADR-002 评分引擎与政策因子

- 状态：Accepted
- 日期：2026-03-28

## 背景
A 股基金受政策与制度环境影响显著。仅依赖传统收益/风险因子会遗漏“政策驱动”特征。

## 决策
采用两层评分模型：

1. Base Score（基础层）
- 收益能力
- 风险控制
- 风险收益比
- 稳定性
- 成本效率
- 流动性
- 规模与存续质量

2. Policy Overlay（政策层）
- 政策支持强度
- 政策执行进度
- 监管约束风险（分数越高越安全）

## 公式
- `PolicyScore = 0.4 * support + 0.3 * execution + 0.3 * regulation_safety`
- `FinalScore = BaseScore * (1 - overlay_weight) + PolicyScore * overlay_weight`

## Overlay 权重模板
| 基金类型 | 保守 | 均衡 | 进取 |
|---|---:|---:|---:|
| equity | 0.10 | 0.15 | 0.20 |
| etf_theme | 0.15 | 0.20 | 0.25 |
| bond | 0.05 | 0.08 | 0.10 |

## 风险控制
- Overlay 权重上限 0.25。
- Policy 事件只在发布时间后可计入评分。
- 输出解释必须包含政策因子是否为主要加分/扣分来源。
